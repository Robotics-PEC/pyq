## About The Project

PEC Paper is an open source project started to help PEC students find previous year paper in one place, hence helping them to prepare for exams.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

This project requires a working [Node.js](https://nodejs.org/en/) installation with NPM and [git](https://git-scm.com/downloads)

- To install npm run:
  ```sh
  npm install npm@latest -g
  ```

### Run project locally

1. [Fork](https://help.github.com/articles/fork-a-repo/) this repository to your own GitHub account, and then [clone](https://help.github.com/articles/cloning-a-repository/) it to your local device.
   ```sh
   git clone https://github.com/your_username_/PEC-Previous-Year-Papers.git
   ```
2. Install the dependencies with:
   ```sh
   npm install
   ```
3. Start development server:
   ```sh
   npm run start
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
4. Building
   You can build with:
   ```sh
   npm run build
   ```
5. Testing the production build:
   ```sh
   npm run start
   ```
6. To run test:
   ```sh
   npm run test
   ```
